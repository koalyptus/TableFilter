Filter and sort column with images

This demo shows how to filter and sort a column containing images by using the custom_cell_data delegate and defining a data-tf-sortKey attribute in the cells of the column.

TableFilter automatically detects the presence of the data-tf-sortKey attibute in a cell. If the sort extension is enabled the sort mechanism will sort based the column on the value specified in the data-tf-sortKey attibute.
